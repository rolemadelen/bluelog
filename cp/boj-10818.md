[10818 - 최소, 최대](https://www.acmicpc.net/problem/10818)

N개의 정수가 주어진다. 이때, 최솟값과 최댓값을 구하는 프로그램을 작성하시오.

## C++

```cpp
#include <bits/stdc++.h> 
using namespace std;

int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);
  int min = 1000001, max = -1000001;
  int n; cin >> n;
  for(int i=0; i<n; ++i) {
    int t; cin >> t;
    min = (t < min) ? t : min;
    max = (t > max) ? t : max;
  }
  cout << min << ' ' << max << endl;
  return 0;
}
```
