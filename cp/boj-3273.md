[3273 - 두 수의 합](https://www.acmicpc.net/problem/3273)

n개의 서로 다른 양의 정수 a1, a2, ..., an으로 이루어진 수열이 있다. ai의 값은 1보다 크거나 같고, 1000000보다 작거나 같은 자연수이다. 자연수 x가 주어졌을 때, ai + aj = x (1 ≤ i < j ≤ n)을 만족하는 (ai, aj)쌍의 수를 구하는 프로그램을 작성하시오.

## C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);

  int n, x, t, cnt=0;
  vector<int> a;
  scanf("%d", &n);
  for(int i=0; i<n; ++i) {
    scanf("%d", &t);
    a.push_back(t);
  }
  scanf("%d", &x);

  // O(n log n)
  sort(a.begin(), a.end());

  // O(n^2)
  for(int i=0; i<n-1; ++i) {
    for(int j=i+1; j<n; ++j) {
      if(a[i] + a[j] == x) ++cnt;
      else if(a[i] + a[j] > x) break;
    }
  }

  printf("%d\n", cnt);
  return 0;
}
```
