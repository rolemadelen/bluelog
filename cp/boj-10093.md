[10093 - 숫자](https://www.acmicpc.net/problem/10093)

두 양의 정수가 주어졌을 때, 두 수 사이에 있는 정수를 모두 출력하는 프로그램을 작성하시오.

## C++


```cpp
#include <bits/stdc++.h> 
using namespace std;
int main(void) {
  uint64_t a,b,c;
  scanf("%llu %llu", &a, &b);
  if (a > b) swap(a,b);
  c = b-a-1;
  printf("%llu\n", (a==b) ? 0 : c);
  for(uint64_t i=a+1; i<b; ++i) printf("%llu ", i);
  return 0;
}
```
