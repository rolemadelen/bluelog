[2439: 별 찍기 - 2](https://www.acmicpc.net/problem/2439)

첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 오른쪽 정렬로 찍는 문제

## C++

```cpp
#include <bits/stdc++.h> 
using namespace std;
int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);
  int n; cin >> n;

  for(int i=0;i <n; ++i) {
    for(int j=n; j>i+1; --j)
      cout << " ";
    for(int j=0; j<i+1; ++j)
      cout << "*";
    cout << endl;
  }

  return 0;
}
```
