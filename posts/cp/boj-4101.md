[4101 - 크냐?](https://www.acmicpc.net/problem/4101)

두 양의 정수가 주어졌을 때, 첫 번째 수가 두 번째 수보다 큰지 구하는 프로그램을 작성하시오.

## C++

```cpp
#include <bits/stdc++.h> 
using namespace std;
int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);
  int a,b;
  while(1) {
    cin >> a >> b;
    if(!a && !b) break;
    cout << ((a>b) ? "Yes" : "No") << endl;
  }

  return 0;
}
```
