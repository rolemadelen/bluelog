[5338 - 마이크로소프트 로고](https://www.acmicpc.net/problem/5338)

마이크로소프트 로고를 예제 출력처럼 출력하는 프로그램을 작성하시오.

## C++
```cpp
#include <bits/stdc++.h> 
using namespace std;
int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);
  cout<<"       _.-;;-._" << endl;
  cout<<"'-..-'|   ||   |"<<endl;
  cout<<"'-..-'|_.-;;-._|"<<endl;
  cout<<"'-..-'|   ||   |"<<endl;
  cout<<"'-..-'|_.-''-._|"<<endl;

  return 0;
}
```
