[2558: A+B - 2](https://www.acmicpc.net/problem/2558)

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

## C++

```cpp
#include <bits/stdc++.h> 
using namespace std;

int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);

  int a, b;
  cin >> a >> b;
  cout << a + b << endl;

  return 0;
}
```
