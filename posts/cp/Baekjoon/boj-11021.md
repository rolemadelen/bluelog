[11021 - A+B - 7](https://www.acmicpc.net/problem/11021)

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.


## C++
```cpp
#include <bits/stdc++.h> 
using namespace std;

int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);

  int t, a, b;
  scanf("%d", &t);

  for(int i=1; i<=t; ++i) {
    scanf("%d %d", &a, &b);
    printf("Case #%d: %d\n", i, a+b);
  }

  return 0;
}
```
