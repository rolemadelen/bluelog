## 문제

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

## C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);

  int a, b;
  cin >> a >> b;
  cout << a+b;

  return 0;
}
```

## 시간 복잡도
- 입출력 - O(1)
- 최종 시간 복잡도: **O(1)**

## 공간 복잡도
- 공간 복잡도: **O(1)**
