[1330: 두 수 비교하기](https://www.acmicpc.net/problem/1330)

두 정수 A와 B가 주어졌을 때, A와 B를 비교하는 프로그램을 작성하시오.

## C++

```cpp
#include <bits/stdc++.h> 
using namespace std;
int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);

  int a, b;
  cin >> a >> b;
  cout << ((a==b) ? "==" : (a>b) ? ">" : "<") << endl;

  return 0;
}
```
