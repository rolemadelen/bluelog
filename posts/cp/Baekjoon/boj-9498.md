[9498 - 시험 성적](https://www.acmicpc.net/problem/9498)

시험 점수를 입력받아 90 ~ 100점은 A, 80 ~ 89점은 B, 70 ~ 79점은 C, 60 ~ 69점은 D, 나머지 점수는 F를 출력하는 프로그램을 작성하시오.

## C++

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(void) {
  int s; cin>>s;
  if(s>=90) cout<<'A';
  else if(s>=80) cout<<'B';
  else if(s>=70) cout<<'C';
  else if(s>=60) cout<<'D';
  else cout<<'F';
  return 0;
}
```
