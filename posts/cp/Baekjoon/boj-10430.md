[10430 - 나머지](https://www.acmicpc.net/problem/10430)

(A+B)%C는 ((A%C) + (B%C))%C 와 같을까?

(A×B)%C는 ((A%C) × (B%C))%C 와 같을까?

세 수 A, B, C가 주어졌을 때, 위의 네 가지 값을 구하는 프로그램을 작성하시오.

## C++

```cpp
#include <bits/stdc++.h> 
using namespace std;
int main(void) {
  ios::sync_with_stdio(0);
  cin.tie(0);

  int a, b, c;
  cin >> a >> b >> c;
  cout << (a+b)%c << endl;
  cout << ((a%c) + (b%c))%c << endl;
  cout << (a*b)%c << endl;
  cout << ((a%c) * (b%c))%c << endl;

  return 0;
}
```
