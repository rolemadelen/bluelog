[11726 - 2×n 타일링](https://www.acmicpc.net/problem/11726)

2×n 크기의 직사각형을 1×2, 2×1 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.

## C++

```cpp
#include <bits/stdc++.h> 
using namespace std;
int main(void) {
  const int MOD = 10007;
  int n;
  cin >> n;
  vector<unsigned> v(6, 0);
  v[1] = 1;
  v[2] = 2;
  v[3] = 3;
  v[4] = 5;
  v[5] = 8;
  for(int i=6; i<=n; ++i) 
    v.push_back((v[i-2]%MOD + v[i-1]%MOD)%MOD);

  cout << v[n] << endl;

  return 0;
}
```
